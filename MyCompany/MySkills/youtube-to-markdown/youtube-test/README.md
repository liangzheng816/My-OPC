# youtube-test — example outputs

End-to-end example outputs from this skill, run against [Lenny's Podcast — "How Anthropic's product team moves faster than anyone else" (Cat Wu)](https://www.youtube.com/watch?v=PplmzlgE0kg) — an ~85-minute interview, 2,356 transcript snippets.

These are real artifacts produced by the skill in a Claude Code session. They show what Steps 1, 2a, and 2b produce in practice. **No API key was used at any point** — the LLM steps were performed by the host agent.

## What each path demonstrates

| Path | Skill step | Notes |
|---|---|---|
| `*.transcript.md` | Step 1 (deterministic) | Generated by `bun scripts/main.ts transcript <url>`. No API key required. |
| `*.summary.md` | Step 2a (agent) | Host agent applied [`prompts/summary.md`](../prompts/summary.md) to the transcript content. |
| `<slug>/00-index.md` + `NN-*.md` | Step 2b (agent) | Outline → per-chapter generation (parallel-friendly) → deterministic index. |
| `infographic/...` | Downstream | Each per-chapter MD (or the single summary) is a clean input to a chapter-infographic step. The `infographic/` folder shows the result of feeding `summary.md` into a separate `baoyu-infographic` skill. Not part of this skill. |

## Reproduce

From a terminal:

```bash
SKILL=~/.claude/skills/youtube-to-markdown/scripts/main.ts

npx -y bun "$SKILL" transcript \
    "https://www.youtube.com/watch?v=PplmzlgE0kg" \
    --out ./youtube-test
```

Step 2 (summarize) is performed by the host agent in a Claude Code / Codex session — paste the URL with intent ("give me a single summary" or "break this into chapter notes for infographics") and the agent will follow the workflow in [SKILL.md](../SKILL.md).
