## Image Specifications

- **Type**: Infographic poster
- **Layout**: bento-grid
- **Style**: bold-graphic (pop-art / comic-book variant — halftone dots, bold black outlines, primary-color blocks)
- **Aspect Ratio**: 9:16 (portrait)
- **Language**: English

## Core Principles

- Follow the layout structure precisely for information architecture.
- Apply style aesthetics consistently throughout.
- Keep information concise, highlight keywords and core concepts.
- Use ample whitespace for visual clarity within each cell.
- Maintain clear visual hierarchy with one hero cell.

## Layout Guidelines (bento-grid)

- Grid of rectangular cells with mixed sizes (1×1, 2×1, 1×2, 2×2). No strict symmetry.
- Cells separated by clear thick black borders.
- Place a single **hero cell** at the top occupying the full width — this is the headline metric. The hero cell must visually dominate.
- Below the hero, organize 7 supporting cells in a varied bento arrangement. Aim for: 2 cells in one row, then a wide cell, then a 2×1 row, then 2 cells, then a wide footer cell.
- Each cell has its own small headline + a tiny visual element (icon, mini-chart, or diagram fragment).
- Consistent padding inside cells. No cell should feel cramped.

## Style Guidelines (bold-graphic, pop-art variant)

- **Palette**: bold primaries — red, yellow, blue, black, white. Spot accent of orange or magenta. Pop-art halftone dot patterns as backgrounds for selected cells.
- **Outlines**: thick, hand-drawn black outlines around every shape and cell.
- **Typography**: comic-book impact lettering for the main title and any "POW!"-style emphasis. Clean readable sans-serif (in the comic-graphic family) for body labels.
- **Emphasis devices**: starbursts, action lines, occasional onomatopoeic accents (e.g. "POW", "BAM") used sparingly.
- **Halftone**: use Ben-Day style halftone dots in 2–3 cell backgrounds for texture; do not over-apply.
- **Mood**: high-energy, attention-grabbing, punchy — like a vintage comic-book cover crossed with a modern data poster.
